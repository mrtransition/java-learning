学习路径建议

🎯 学习路线
  • 初学者：按 LEVEL 1 → LEVEL 2 → LEVEL 3 顺序学习，每学完一节动手敲代码
  • 有编程基础：可从 LEVEL 2 开始，遇到陌生概念再回看 LEVEL 1
  • 每节目标：理解概念 → 阅读代码 → 完成练习 → 进入下一节
  • 学习节奏：建议每周 2-3 节，确保每节练习都亲手完成

关于本教程

面向有基本编程概念的读者，以 Java 8 为基础。新增特性均标注版本号 Java 8+。所有示例均遵循 Java 官方规范，可直接在 JDK 8+ 环境中编译运行。

💡 高效学习法
  1. 先读后练：阅读概念讲解，理解设计意图
  2. 亲手敲代码：不要复制粘贴，逐行输入加深记忆
  3. 修改参数：修改代码中的值观察输出变化
  4. 完成练习：每节后的练习题帮助巩固知识点
  5. 回顾总结：学完一节后用自己的话复述要点

环境准备

  1. 下载安装 JDK 8（或 JDK 17+ Java 11+）
  2. 验证安装：java -version 和 javac -version
  3. 用任意文本编辑器编写 .java 文件
  4. 编译运行：javac Main.java && java Main
