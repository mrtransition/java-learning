🗄️ 数据库访问

企业级应用几乎都需要数据库。Spring Boot 提供了 JPA 和 JDBC 两种主流方案。

Spring Data JPA

JPA 是 Java 官方的 ORM 规范，Spring Data JPA 是其最便捷的实现：

// 1. 实体类
@Entity
@Table(name = "users")
public class User {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @NotBlank
    private String name;

    @Email
    private String email;

    // getter / setter（必写）
}

// 2. Repository 接口 — 无需实现类
public interface UserRepository extends JpaRepository<User, Long> {
    // 根据方法名自动生成查询
    List<User> findByName(String name);
    Optional<User> findByEmail(String email);
    List<User> findByNameContainingIgnoreCase(String keyword);
    boolean existsByEmail(String email);

    // 自定义 JPQL
    @Query("SELECT u FROM User u WHERE u.email LIKE %:domain")
    List<User> findByEmailDomain(@Param("domain") String domain);

    // 原生 SQL 查询
    @Query(value = "SELECT * FROM users WHERE age > :min",
           nativeQuery = true)
    List<User> findByAgeGreaterThan(@Param("min") int min);
}

// 3. 使用
@Service
public class UserService {
    private final UserRepository repository;

    public UserService(UserRepository repository) {
        this.repository = repository;
    }

    public User createUser(User user) {
        return repository.save(user);
    }

    public List<User> searchByName(String name) {
        return repository.findByNameContainingIgnoreCase(name);
    }
}

实体关系映射

@Entity
public class Order {
    @Id @GeneratedValue private Long id;

    @ManyToOne(fetch = FetchType.LAZY)   // 多对一，延迟加载
    @JoinColumn(name = "user_id")
    private User user;

    @OneToMany(mappedBy = "order", cascade = CascadeType.ALL, orphanRemoval = true)
    private List<OrderItem> items;

    @Enumerated(EnumType.STRING)
    private OrderStatus status;
}

public enum OrderStatus { PENDING, PAID, SHIPPED, DELIVERED, CANCELLED }

事务管理

@Service
public class PaymentService {
    @Transactional  // 失败时自动回滚
    public void transfer(Long fromId, Long toId, double amount) {
        Account from = accountRepo.findById(fromId).orElseThrow();
        Account to = accountRepo.findById(toId).orElseThrow();
        from.withdraw(amount);
        to.deposit(amount);
        // 任意一步抛异常，全部回滚
    }

    @Transactional(readOnly = true)  // 只读事务优化
    public Account getAccount(Long id) {
        return accountRepo.findById(id).orElseThrow();
    }
}

Flyway — 数据库版本管理

<!-- pom.xml -->
<dependency>
    <groupId>org.flywaydb</groupId>
    <artifactId>flyway-core</artifactId>
</dependency>

// application.yml
spring:
  flyway:
    enabled: true
    locations: classpath:db/migration

// src/main/resources/db/migration/V1__create_users.sql
CREATE TABLE users (
    id    BIGINT AUTO_INCREMENT PRIMARY KEY,
    name  VARCHAR(100) NOT NULL,
    email VARCHAR(255) NOT NULL UNIQUE
);

// V2__add_age_column.sql
ALTER TABLE users ADD COLUMN age INT;
💡 JPA 最佳实践
  • 所有关联使用 FetchType.LAZY，避免 N+1 问题
  • 实体类必须写 equals() 和 hashCode()（用 ID 实现）
  • 生产环境 ddl-auto 设为 validate 或 none，用 Flyway 管理 DDL
  • 复杂查询用 @Query + JPQL，避免过长的 find 方法名
