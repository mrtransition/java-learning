📦 集合框架

Java Collections Framework 提供了一套统一的接口和实现类。核心接口：

接口特点常用实现
List有序、可重复ArrayList, LinkedList
Set不可重复HashSet, TreeSet, LinkedHashSet
Map键值对HashMap, TreeMap, LinkedHashMap
Queue队列（FIFO）ArrayDeque, LinkedList

List

List<String> fruits = new ArrayList<>();

fruits.add("Apple");
fruits.add("Banana");
fruits.add("Cherry");
fruits.add("Apple");    // List 允许重复

System.out.println(fruits.get(0));        // Apple
System.out.println(fruits.size());         // 4
System.out.println(fruits.contains("Banana"));  // true

// 遍历
for (String fruit : fruits) {
    System.out.println(fruit);
}

Set

Set<Integer> numbers = new HashSet<>();

numbers.add(3);
numbers.add(1);
numbers.add(2);
numbers.add(3);   // 重复元素被忽略

System.out.println(numbers.size());     // 3
System.out.println(numbers.contains(2));  // true

// HashSet 无序，如需有序：
Set<Integer> sorted = new TreeSet<>(numbers);
Set<Integer> insertionOrder = new LinkedHashSet<>(numbers);

Map

Map<String, Integer> scores = new HashMap<>();

scores.put("Alice", 95);
scores.put("Bob", 87);
scores.put("Charlie", 92);

System.out.println(scores.get("Alice"));       // 95
System.out.println(scores.getOrDefault("David", 0));  // 0

// 遍历
for (Map.Entry<String, Integer> entry : scores.entrySet()) {
    System.out.println(entry.getKey() + ": " + entry.getValue());
}

// Java 8+ 便捷方法
scores.merge("Bob", 5, Integer::sum);  // Bob 变为 92
scores.computeIfAbsent("Eve", k -> 88);  // 键不存在时计算

Collections 工具类

List<String> list = new ArrayList<>(Arrays.asList("c", "a", "b"));

Collections.sort(list);              // [a, b, c]
Collections.reverse(list);           // [c, b, a]
Collections.shuffle(list);           // 随机打乱

List<String> immutable = Collections.unmodifiableList(list);
// immutable.add("x");  // ❌ 运行时异常

// List.of / Set.of / Map.of (Java 9+) 创建不可变集合
List<String> fixed = Collections.unmodifiableList(
    Arrays.asList("x", "y", "z"));
Set<Integer> nums = Collections.unmodifiableSet(
    new HashSet<>(Arrays.asList(1, 2, 3)));
Map<String, Integer> ages = new HashMap<>();
ages.put("Alice", 30);
ages.put("Bob", 25);
ages = Collections.unmodifiableMap(ages);
