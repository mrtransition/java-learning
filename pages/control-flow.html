if / else if / else

int score = 85;

if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 75) {
    System.out.println("良好");     // 输出
} else if (score >= 60) {
    System.out.println("及格");
} else {
    System.out.println("不及格");
}

switch 语句

int day = 3;
String dayName;

switch (day) {
    case 1: dayName = "周一"; break;
    case 2: dayName = "周二"; break;
    case 3: dayName = "周三"; break;
    default: dayName = "未知";
}
System.out.println(dayName);  // 周三

switch 表达式 Java 14+

增强 switch 支持箭头语法，无需 break，可直接赋值：

String day = "MONDAY";

int result = switch (day) {
    case "MONDAY", "FRIDAY" -> 1;
    case "WEDNESDAY" -> 2;
    default -> {
        System.out.println("其他");
        yield 0;      // 多行分支用 yield 返回值
    }
};
System.out.println(result);  // 1

for 循环

// 传统 for
for (int i = 0; i < 5; i++) {
    System.out.print(i + " ");   // 0 1 2 3 4
}

// 增强 for (for-each)
int[] numbers = {10, 20, 30};
for (int n : numbers) {
    System.out.println(n);
}

while / do-while

// while — 先判断再执行
int i = 0;
while (i < 3) {
    System.out.println(i);
    i++;
}

// do-while — 至少执行一次
int j = 0;
do {
    System.out.println("至少一次");
} while (j > 0);
⚠️ 初学者常见错误
  • if 后加分号：if (x > 0); 分号结束了语句，后面的代码块总会执行
  • == 比较字符串：字符串比较用 .equals()，== 比较的是引用地址
  • switch 忘记 break：传统 switch 会穿透到下一个 case，务必加 break
  • 无限循环：确保循环条件最终会变为 false，避免 while (true) 没有出口

break 与 continue

for (int i = 0; i < 10; i++) {
    if (i == 3) continue;   // 跳过 3，继续循环
    if (i == 7) break;      // 到 7 时退出循环
    System.out.print(i + " ");  // 0 1 2 4 5 6
}
📌 本节要点
  • 分支结构有 if/else 和 switch 两种，switch 表达式 (Java 14+) 更简洁安全
  • 三种循环中 for 适合已知次数，while 适合条件控制，do-while 保证至少执行一次
  • break 退出整个循环，continue 跳过当前轮次继续下一轮
  • Switch 表达式用 -> 箭头语法，不需要 break，多行语句用 yield 返回值
✏️ 动手练习
  1. 用 if/else 实现一个成绩等级判断：90+ 为 A，80+ 为 B，70+ 为 C，60+ 为 D，其余为 F
  2. 用 switch 表达式将数字 1-7 转换为中文星期几
  3. 用 for 循环打印 1 到 100 之间的所有偶数
  4. 用 while 循环计算 1+2+3+...+100 的和
  5. 输出 1-20，但跳过 5 的倍数，到 17 停止（综合使用 continue 和 break）
