🔤 变量与数据类型

基本数据类型

Java 有 8 种基本类型（primitive types）：

类型大小范围/示例
byte1 字节-128 ~ 127
short2 字节-32,768 ~ 32,767
int4 字节-2³¹ ~ 2³¹-1
long8 字节结尾加 L：100L
float4 字节结尾加 f：3.14f
double8 字节默认小数类型：3.14
char2 字节单个字符：'A'
boolean1 位true / false
public class Basics {
    public static void main(String[] args) {
        // 基本类型声明
        int age = 25;
        double price = 19.99;
        boolean isActive = true;
        char grade = 'A';
        long population = 8_000_000_000L;

        System.out.println("Age: " + age);
        System.out.println("Price: " + price);
        System.out.println("Active: " + isActive);
    }
}

引用类型与 String

String 是 Java 中最常用的引用类型，不是基本类型。

public class StringExample {
    public static void main(String[] args) {
        String name = "Java";
        String greeting = "Hello, " + name + "!";

        // 常用方法
        System.out.println(greeting);                 // Hello, Java!
        System.out.println(name.length());            // 4
        System.out.println(name.toUpperCase());       // JAVA
        System.out.println(name.contains("va"));   // true
        System.out.println(String.format("值: %.2f", 3.14159));
    }
}

类型转换

自动转换（隐式）

小范围类型可自动转为大范围类型：byte → short → int → long → float → double

int i = 100;
long l = i;      // 自动转换
double d = i;    // 自动转换

强制转换（显式）

大范围转小范围可能丢失精度：

double pi = 3.14159;
int truncated = (int) pi;  // 3，小数部分被截断

long big = 1_000_000_000_000L;
int small = (int) big;    // 溢出，结果不正确
⚠️ 注意 强制类型转换可能导致数据溢出或精度丢失，使用前请确认值在目标类型范围内。

var 关键字（Java 10+）

局部变量可用 var 让编译器推断类型：

var message = "Hello";      // 等同于 String message
var count = 42;              // 等同于 int count
var list = new ArrayList<String>();  // ArrayList<String>
// var x = null;           // ❌ 编译错误，无法推断类型
