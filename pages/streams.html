🌊 Lambda 与 Stream API

Lambda 表达式

Lambda 提供简洁的匿名函数语法，核心格式：(参数) -> 表达式

// 传统匿名内部类
Runnable r1 = new Runnable() {
    @Override
    public void run() {
        System.out.println("Hello");
    }
};

// Lambda 等价写法
Runnable r2 = () -> System.out.println("Hello");

// 带参数的 Lambda
Comparator<String> cmp = (a, b) -> a.length() - b.length();

List<String> names = Arrays.asList("Java", "Python", "C");
names.sort((a, b) -> Integer.compare(a.length(), b.length()));
System.out.println(names);  // [C, Java, Python]

函数式接口

Lambda 配合函数式接口（只有一个抽象方法）使用：

接口输入输出示例
Predicate<T>Tbooleans -> s.isEmpty()
Function<T,R>TRs -> s.length()
Consumer<T>Tvoidx -> System.out.println(x)
Supplier<T>无T() -> Math.random()
// 自定义函数式接口
@FunctionalInterface
interface Calculator {
    int calculate(int x, int y);
}

Calculator add = (a, b) -> a + b;
Calculator multiply = (a, b) -> a * b;

System.out.println(add.calculate(3, 4));       // 7
System.out.println(multiply.calculate(3, 4));  // 12

Stream API

Stream 用于对集合进行声明式处理（过滤、映射、归约等）。

创建 Stream

List<String> items = Arrays.asList("apple", "banana", "cherry");
// List.of(...) (Java 9+) 可替代 Arrays.asList

// 从集合创建
Stream<String> stream1 = items.stream();

// 从数组创建
Stream<Integer> stream2 = Arrays.stream(new Integer[]{1, 2, 3});

// 直接创建
Stream<String> stream3 = Stream.of("a", "b", "c");

// 无限流
Stream<Double> randoms = Stream.generate(Math::random);
Stream<Integer> natural = Stream.iterate(0, n -> n + 1);

中间操作（返回新 Stream）

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5, 6);

List<Integer> result = numbers.stream()
    .filter(n -> n % 2 == 0)           // 过滤偶数: [2, 4, 6]
    .map(n -> n * n)                    // 平方: [4, 16, 36]
    .sorted((a, b) -> b - a)           // 降序: [36, 16, 4]
    .limit(2)                           // 取前2: [36, 16]
    .collect(Collectors.toList());

System.out.println(result);  // [36, 16]

终端操作

List<String> words = Arrays.asList("hello", "world", "java");

// forEach — 遍历
words.stream().forEach(System.out::println);

// collect — 收集到集合
List<String> upper = words.stream()
    .map(String::toUpperCase)
    .collect(Collectors.toList());     // [HELLO, WORLD, JAVA]

// reduce — 归约
int sum = Arrays.asList(1, 2, 3, 4).stream()
    .reduce(0, Integer::sum);          // 10

// 统计
long count = words.stream()
    .filter(s -> s.length() > 3)
    .count();                          // 3

boolean any = words.stream()
    .anyMatch(s -> s.startsWith("h"));  // true

Optional

Optional 用于优雅处理可能为 null 的值，避免 NullPointerException：

// 创建
Optional<String> empty = Optional.empty();
Optional<String> present = Optional.of("hello");
Optional<String> nullable = Optional.ofNullable(mightBeNull);

// 安全使用
String result = present
    .map(String::toUpperCase)
    .filter(s -> s.length() > 2)
    .orElse("default");          // "HELLO"

// 链式调用
String name = findUser(userId)
    .flatMap(User::getMiddleName)
    .orElseThrow(() -> new NoSuchElementException("无中间名"));
⚠️ 注意
  • Stream 是惰性的——没有终端操作时中间操作不会执行
  • Stream 只能消费一次，重复使用会抛异常
  • 避免在 Lambda 中修改外部变量——变量应实际上是 final 的
